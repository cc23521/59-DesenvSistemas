A tag <html> indica a raiz de todo documento HTML e compreende outras tags, como <head> e <body>. <head> é uma tag utilizada para includir metadados relevantes para a página, como o título da aba que é definido por <title>. A tag <body> inclui todas as tags cujos conteúdos serão renderizados pelo navegador (<p>, <h>, <img>, <ul>, <ol>, <a> etc.) ou que servem para a organização do conteúdo de uma página (<div>, <header>, <main>, <footer>, <nav> etc.).
